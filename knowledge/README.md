# Knowledge System

This folder will eventually store categorized knowledge to make Jarvis context-aware of the user's life and environment.

## Structure

* **personal knowledge**: Information about the user's life, preferences, and personal facts.
* **technical knowledge**: Code snippets, documentation, and technical stack details for the user's projects.
* **project knowledge**: Deep context on the user's active projects, goals, and architectures.
* **device knowledge**: Information on the user's connected devices, their states, and how to interact with them.
* **life knowledge**: General world knowledge customized and fact-checked specifically for the user's worldview.

These files will be dynamically embedded into the vector store or loaded dynamically into agent contexts.
